# MANIT App Store Web Application

A web-based application platform that showcases and categorizes various apps across different domains including Education, Gaming, Food Delivery, and Social Media.

## 📱 Overview

MANIT App Store is a responsive web application built with HTML, CSS, and JavaScript that provides users with a centralized platform to discover and explore applications across multiple categories. The application features an intuitive user interface with authentication capabilities and category-based browsing.

## ✨ Features

- **User Authentication**
  - Login page with username and password validation
  - Sign-up page for new user registration

- **Category Browsing**
  - **Education**: Educational apps and learning platforms
  - **Gaming**: Gaming applications and entertainment
  - **Food/Savory**: Food delivery and restaurant apps
  - **Social Media**: Social networking applications

- **Responsive Design**
  - Mobile-friendly interface
  - Smooth navigation between categories
  - Interactive hover effects and animations

- **User-Friendly Interface**
  - Clean and intuitive navigation menu
  - Category cards with background images
  - Search functionality
  - Quick access to different app categories

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side installation required

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/App-Store-Web-Application.git
   cd App-Store-Web-Application
   ```

2. **Open the application**
   - Open `prologin.html` in your web browser to start with the login page
   - Or open `prohomepage1.html` to view the main category page directly

3. **Explore categories**
   - Click on any category card to browse apps in that category

## 📖 Usage

### Navigation Flow

1. **Landing Page** (`prologin.html`)
   - Enter your credentials to log in
   - New users can click "Sign Up" to create an account

2. **Main Dashboard** (`prohomepage1.html`)
   - View all available app categories
   - Click on any category to explore apps

3. **Category Pages** (`pro2.html`, `pro3.html`, `pro4.html`, `pro5.html`)
   - Browse apps within your selected category
   - View app details and descriptions
